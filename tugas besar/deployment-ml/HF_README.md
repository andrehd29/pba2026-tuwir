# 🎵 Analisis Sentimen Ulasan Spotify — ML Model

Aplikasi ini menganalisis sentimen ulasan pengguna Spotify dalam Bahasa Indonesia menggunakan model Machine Learning yang dibangun dengan **PyCaret AutoML**.

## Model
- **Algoritma:** SVM, Multinomial Naive Bayes, Decision Tree
- **Dataset:** 100.000 ulasan Spotify dari Google Play Store
- **Kelas:** Positif, Netral, Negatif
